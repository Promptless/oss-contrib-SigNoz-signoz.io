SigNoz can be installed using marketplaces options providing one-click installations. This documentation provides an overview of these installation methods.

<KeyPointCallout title="Foundry-powered marketplace templates" defaultCollapsed={false}>
SigNoz maintains upstream templates on these platforms, powered by <a href="https://github.com/SigNoz/foundry" target="_blank" rel="noopener noreferrer nofollow">Foundry</a> castings:

- **Railway** — Deploy from the <a href="https://github.com/SigNoz/foundry/tree/main/docs/examples/railway/template" target="_blank" rel="noopener noreferrer nofollow">Railway template casting</a>. One-click deploy via Railway's template marketplace.
- **Coolify** — Deploy from the <a href="https://github.com/SigNoz/foundry/tree/main/docs/examples/coolify/stack" target="_blank" rel="noopener noreferrer nofollow">Coolify stack casting</a>. Available as a stack in Coolify's service catalog.
</KeyPointCallout>
